Abstract
Wave massive MIMO systems are among the promising technologies for the next generation of wireless communication systems to cover the requirement for high data rates. In such systems, hybrid precoders are preferred to digital beamforming systems as they offer Low-power-consumption with an acceptable performance compromise. This paper proposes a novel hybrid precoders for partially connected architectures that use low-resolution phase shifters in the mm-Wave massive MIMO systems. The proposed algorithm uses cross-entropy to iteratively calculate probability distributions and select the best precoder design. Simulation results show that the proposed algorithm outperforms existing solutions offered for the partially connected architecture and closes the gap to fully connected architectures.
